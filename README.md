# 加拿大10年级数感训练

打开 `index.html` 即可使用，不需要安装依赖。

## 内容结构

- 10 天训练计划
- 8 个加拿大 Grade 10 数感模块
- 每天每个模块 8 道填空题，共 640 道题
- 覆盖线性关系、方程组、多项式、指数根式、二次关系、直角三角、金融比例、数据概率
- 学生可以填写答案、标注不会
- 教师账号为 `Elven Zeng`，可查看答案、参考答案、批改对错、写批注
- 第二天会提醒学生订正前一天被批为错误的题
- 页面会显示每日统计、总进度和成就徽章

## 数据说明

练习记录保存在当前浏览器的本地存储中。若学生和老师不在同一台设备或同一个浏览器，可在教师页面使用“导出记录”和“导入记录”同步数据。
